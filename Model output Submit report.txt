
# **YOLO11n Object Detection Model Report**

## **1. Overview**

The YOLO11n object detection model was trained on approximately **5000 images** to detect two classes: **customer** and **employee**. The model was evaluated using a separate validation dataset consisting of **655 images**, ensuring unbiased performance measurement.

---

## **2. Dataset Details**

* **Validation Images:** 655
* **Total Instances:** 1810
* **Customer Instances:** 1414
* **Employee Instances:** 396

Each instance represents a labeled object in an image. Multiple instances can exist within a single image.

---

## **3. Model Architecture**

* **Model:** YOLO11n (Nano version)
* **Total Layers:** 100
* **Parameters:** 2,582,542
* **Computational Cost:** 6.3 GFLOPs

The model is lightweight and optimized for real-time performance.

---

## **4. Performance Metrics**

* **Precision:** 95.1%
* **Recall:** 97.3%
* **mAP@0.5:** 98.1%
* **mAP@0.5:0.95:** 70.3%
* **F1-Score:** 96.2%

### Explanation:

* **Precision** measures how many predicted objects are correct.
* **Recall** measures how many actual objects are detected.
* **mAP@0.5** evaluates detection accuracy at IoU threshold 0.5.
* **mAP@0.5:0.95** evaluates stricter bounding box accuracy across multiple IoU thresholds.
* **F1-score** balances precision and recall.

---

## **5. Class-wise Performance**

### Customer:

* Precision: 97.2%
* Recall: 96.8%
* mAP@0.5: 98.7%
* mAP@0.5:0.95: 70.1%

### Employee:

* Precision: 93.1%
* Recall: 97.7%
* mAP@0.5: 97.6%
* mAP@0.5:0.95: 70.6%

---

## **6. Inference Speed**

* **Preprocessing Time:** 0.1 ms/image
* **Inference Time:** 0.7 ms/image
* **Postprocessing Time:** 0.4 ms/image

**Total Time per Image:** ~1.2 ms
**Estimated Speed:** ~833 FPS

The model is capable of real-time detection.

---

## **7. Evaluation Methodology**

The model predictions are evaluated using **Intersection over Union (IoU)**.

* Predictions are compared with ground truth bounding boxes
* Based on overlap, detections are classified as:

  * True Positive (TP)
  * False Positive (FP)
  * False Negative (FN)

These values are used to compute Precision, Recall, and mAP.

---

## **8. Conclusion**

The YOLO11n model demonstrates **high detection accuracy and efficiency**. With an mAP@0.5 of **98.1%** and strong recall of **97.3%**, the model effectively identifies objects in images. The mAP@0.5:0.95 score of **70.3%** indicates moderate room for improvement in bounding box precision.

Overall, the model is **fast, reliable, and suitable for real-time applications** such as surveillance and automated customer-employee detection systems.
